Dissertation - Identifying Bars in Galaxies Using Machine Learning (IIT Indore)

Phase 1

  • Developed an automatic method for detecting bar structures in galaxies using YOLO-OBB object detection.
  • Created realistic synthetic galaxy images to train and validate the model.
  • The system accurately extracts physical bar parameters: length, orientation, and shape.
  • Tested the approach on real astronomical images with strong performance.
  • Enables scalable, precise, and interpretable bar identification for large galaxy datasets.

Phase 2

  • Processed Legacy Survey galaxy images from Galaxy Zoo 3 (GZ3), applied pixel-confidence masking to match the Hoyle Bar dataset for cross-validation.
  • Developed and implemented a custom equation to optimize consistency between datasets.
  • Used CLAHE for high-quality PNG extraction from FITS and bilateral symmetry analysis to enhance bar visibility.
  • Annotated and trained YOLOv11 OBB model to identify bar structures, achieving 99.04% accuracy (mAP@0.5) and 96.8% precision.
  • Validated model on the Hoyle dataset with Pearson r = 0.91, MAE = 1.14 Kpc and RMSE = 1.59 Kpc, outperforming Bhambra (2022) and Walmsley (2023).
  • In contrast to the computationally expensive U-Net used by Walmsley (2023), YOLO is faster and demands fewer resources, excelling in tasks.
